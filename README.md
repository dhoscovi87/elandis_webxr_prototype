# Dhosco WebXR Prototype: Elandis Africa

This is a **Tier 3 (VIP)** prototype for Elandis Africa, demonstrating how high-end real estate can be sold using immersive technology.

## 🚀 The Vision
Elandis Africa currently uses static 2D photos to sell villas worth 370M+ FCFA. This prototype shows a path to **Immersive Sales**, where buyers in the global diaspora can walk through a property in 3D directly from their browser.

## 🛠 Tech Stack
- **Three.js**: For the 3D rendering engine.
- **Vite**: For fast development and optimized builds.
- **WebXR API**: Ready for VR/AR headset support.

## 📈 Dhosco Agency Tiers
1. **Tier 1 (Legacy Refresh)**: Web performance & SEO optimization.
2. **Tier 2 (Digital Concierge)**: AI-driven lead qualification & automated property updates.
3. **Tier 3 (VIP Immersive)**: Custom WebXR virtual tours (This Prototype).
